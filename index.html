Internet Basics
HTML is the "glue" that holds it all together.
HTML stands for Hyper Text Markup Language.
Information is transfered from a sever to your desktop via HTTP, which is Hper Text Transfer Protocol.

Tags
We learned about two types of tags. inline and block.
Examples of inline include; Bold, Emphais and Anchor.

Examples of block include; Paragraph and Division

Anchor tags are for adding Links.

Image tags are also for adding images.

Image tags and break tags are examples of "void" tags. (i.e tags with no closing brackets).

We learned about dealing with "white space" through either the break tag or the paragraph tag.
